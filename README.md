# Simple Gear Calculator

a website to visually display gears of different sizes and their speeds, 
support meshing, but also tethering (to copy the speed).

## SImulation modifications
Gear parameters can be modified - size, tooth number, module, and speed.
All gear speeds are calculated from the gear the speed was last modified by user, or by default the first gear with speed 1.

## Alignment
Gears are automatically aligned so teeth do not intersect.

## Technologies
Build with React and js.
